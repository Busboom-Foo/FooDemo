About Insertion Sort

Insertion Sort is a simple sorting algorithm that builds the final sorted array one item at a time. It's much less efficient on large lists than more advanced algorithms such as quicksort, heapsort, or merge sort, but it has several advantages:

  • Simple implementation
  • Efficient for small data sets
  • Adaptive (efficient for datasets that are already substantially sorted)
  • Stable (does not change the relative order of elements with equal keys)
  • In-place (only requires a constant amount O(1) of additional memory space)

Time Complexity:

  • Best Case: O(n) - when the array is already sorted
  • Average Case: O(n²)
  • Worst Case: O(n²) - when the array is sorted in reverse order
